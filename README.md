# scalam
Smart search for upgrade paths

## Installation

On a Debian based system:

``` bash
sudo apt-get install python3-pip
sudo pip3 install ez_setup
```

Install GitPython:

``` bash
git clone https://github.com/gitpython-developers/GitPython
cd GitPython
sudo python3 setup.py install
```

## To run unit tests

``` bash
cd tests
python3 tests.py
```
